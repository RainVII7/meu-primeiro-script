# meu-primeiro-script
Repositório criado para hospedar meu script e código da minha primeira calculadora em python
Instruções de como usar meu script no linux
Para executar o script use o comando ./                 (Caso aparecer a mensagem 'Permission Denied'quando tentar executar o script, será necessário executar o script como super usuário, adicionando 'sudo' antes do comando)
